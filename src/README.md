# A1_Maze_Swift-1

Реализация проекта Maze.

## Install

```
xcodegen generate
make install
```

## About

Проект был разработан **vivastan** и **swissmer**.

Настоящий проект реализует:

1. Прорисовку лабиринта
2. Генерация лабиринта
3. Поиск маршрута для заданных координат
4. Генерация пещеры
